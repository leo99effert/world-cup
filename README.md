## Hur fungerar spelet?

VM 2026 har 48 lag fördelade på fem konfederationer:

- 🌏 **Asien** (AFC/OFC)
- 🌍 **Afrika** (CAF)
- 🌎 **Nord- och Centralamerika** (CONCACAF)
- 🌎 **Sydamerika** (CONMEBOL)
- 🌍 **Europa** (UEFA)

Varje spelare väljer **ett lag per konfederation** – totalt fem lag. Målet är att välja det lag inom varje konfederation som tar sig längst i turneringen.

### Poäng

Du får **ett poäng** för varje lag som tar sig längst i sin konfederation.

### Tiebreaker

Om två spelare har lika många poäng avgörs placeringen av **tiebreakern** – summan av FIFA-rankingen på spelarens fem valda lag. Högre rankingnummer = sämre lag = modigare val = vinner vid lika poäng.
